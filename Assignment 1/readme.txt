Program reads input from "input.txt" file in the same directory

input.txt file contains,
    0, 1 or 2 (respectively for bfs, dfs or dfid)
    m x n maze with food for packman

Gives output in the output.txt file

output.txt file contains 
    Number of states explored in first line.
    Length of the path discovered in second line.
    m x n maze with path printed in 0s.
